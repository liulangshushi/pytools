# arpspoof

```
Usage: arpspoof.py [+]-i <interface> [+]-t <targets> [+]-s <source> [+]-m <mode> [+]-r <transmit>

Options:
  -h, --help    show this help message and exit
  -i INTERFACE  [+]Interface(default eth0)
  -t TARGETS    [+]Target host,split with ','
  -s SOURCE     [+]Source host
  -m MODE       [+]Request(req) or Reply(rep)
  -r TRANSMIT   [+]Transmit(yes or no)
```
